# Liteisle Desktop - 轻屿记

一个基于 Vue 3 + Vite + Electron 的现代化桌面应用程序，采用莫兰迪配色设计，提供专注学习、时间管理和多媒体功能的一体化解决方案。

## ✨ 项目特色

- 🎨 **现代化设计**: 莫兰迪配色 + 无边框窗口设计
- 📚 **专注学习**: 番茄钟、学习统计、活跃度追踪  
- 🎵 **智能音乐播放**: 完整的音乐播放系统，支持拖拽重排、进度控制、歌单管理
- 🗂️ **智能文件管理**: 三级目录、右键菜单、分享系统、回收站、排序搜索
- 📄 **多格式文档查看**: 支持PDF、Word、Excel、PowerPoint、Markdown等格式在线预览
- 📤 **文件传输管理**: 上传下载进度跟踪、分享链接下载、批量操作
- 🔗 **网盘音乐联动**: 网盘文件夹自动映射为音乐播放列表
- ⚙️ **完整设置中心**: 五大分类设置，个性化定制体验
- 🔐 **用户系统**: 简单的登录认证和会话管理

## 🚀 技术栈

- **前端框架**: Vue 3 + TypeScript + Composition API
- **构建工具**: Vite 5.x
- **桌面应用**: Electron 28.x
- **状态管理**: Pinia (统一数据管理)
- **路由管理**: Vue Router (Hash模式)
- **样式框架**: Tailwind CSS 3.x
- **图标库**: Lucide Vue Next
- **拖拽组件**: VueDraggable Next
- **PDF查看**: pdf.js + pdfjs-dist
- **Markdown编辑**: VDitor 富文本编辑器
- **代码高亮**: highlight.js
- **通知提醒**: Vue Toastification
- **活跃度图表**: Vue Activity Calendar
- **包管理**: npm

## 🎨 界面设计

### 布局架构
- **无边框窗口**: 自定义标题栏，集成窗口控制按钮
- **固定侧边栏**: 150px宽度，包含导航和Logo
- **顶部控制栏**: 居中的专注按钮，右侧用户菜单和窗口控制
- **主内容区域**: 响应式布局，右上角圆角设计
- **智能音乐栏**: 底部自适应浮动音乐控制面板

### 色彩系统
- **主背景**: #DDEEEE (liteisle-bg)
- **侧边栏**: #B9E3E3 (liteisle-sidebar)  
- **卡片背景**: #F9FAFA (纯白)
- **强调色**: Teal系列 (#14B8A6)
- **文字色**: Morandi灰色系列

### 交互特性
- **智能响应**: 路由感知的音乐栏自动展开/收起
- **Hover反馈**: 所有可交互元素的悬浮效果和提示
- **过渡动画**: 300ms缓动过渡
- **视觉层次**: 40px圆角 + 阴影系统
- **状态指示**: 当前播放、专注状态的视觉反馈

## 🏗️ 状态管理架构

### Pinia Store设计
- **UIStore**: 全局UI状态管理（侧边栏展开状态等）
- **FocusStore**: 专注模式状态管理（计时器、学习记录）
- **MusicStore**: 音乐播放状态管理（播放器状态、歌单、进度）
- **DriveStore**: 文件管理状态管理（文件列表、目录结构、操作状态）
- **DocsStore**: 文档管理状态管理（文档分类、当前文档、搜索状态）
- **SettingsStore**: 系统设置状态管理（用户偏好、系统配置）

### 数据持久化
- **本地存储**: 使用localStorage保存用户设置和学习数据
- **会话管理**: 登录状态和用户信息持久化
- **状态恢复**: 应用重启后自动恢复用户状态

### 跨组件通信
- **统一数据源**: 所有组件通过Pinia store共享状态
- **响应式更新**: 状态变化自动触发组件重新渲染
- **事件总线**: 复杂交互使用事件系统处理

- **跨平台**: 支持Windows、macOS、Linux多平台构建
- **应用图标**: 自定义应用图标和品牌元素
- **安装选项**: 用户可选择安装路径，支持卸载

### 安装包优化
- **压缩优化**: 使用最新的压缩算法减少安装包体积
- **依赖分析**: 只打包必要的依赖项
- **资源优化**: 图片和静态资源压缩处理
